# babel-plugin-auto-css-modules [![NPM][npm-img]][npm-url] [![Coverage][cov-img]][cov-url]

Automatically import same name CSS modules.

## Installation

**npm**

```bash
npm install babel-plugin-auto-css-modules
```

**yarn**

```bash
yarn add babel-plugin-auto-css-modules
```
